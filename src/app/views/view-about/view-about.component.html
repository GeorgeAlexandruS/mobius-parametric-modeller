What is Möbius Modeller?

Möbius Modeller is an open-source web application for 3D parametric modelling. The app allows users to create modelling procedures using the Obi visual programming language. In the Obi language, your write computer programs by drawing flowcharts and by defining procedures using 'click and fill-in-the-blanks' coding.

Since Mobius Modeller is a web app, you can just go ahead an use it. Just click the icons.

  • It is online, so there is need to install software.
  • It is free, so there is no need to creat an account or to log in.
  • It runs in the browser so there is no need to worry about network speed.
  • Your data never leaves your browser, so there is no need to data privacy and security.

The resulting models can be published as interactive web pages, allowing visitors on the web to explore the models.
